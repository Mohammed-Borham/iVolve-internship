Lab 6: Building and Packaging Java Applications with Gradle


1- Install Gradle.
	# Using Gradle docs https://gradle.org/install/
2- Clone source code https://github.com/Ibrahim-Adel15/build1.git
	$ git clone <url>
3- Run Unit test.
	$ gradle test
4- Build App [ generate artifact in build/libs/ivolve-app.jar ].
	$ gradle build
5- Run App.
	$ gradle run
